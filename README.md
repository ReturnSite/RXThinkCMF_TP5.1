# RXThink_V2.0.0_ThinkPhp5.1

#### 介绍
RXThink是一个开源的权限及内容管理框架，基于ThinkPHP5.0版本开发，提供更方便、更安全的WEB应用开发体验，采用了全新的架构设计和命名空间机制，融合了模块化、驱动化和插件化的设计理念于一体，开启了国内WEB应用傻瓜式开发的新潮流。
